Important Notes

  • USB 3.0 Required: A USB 3.0 port is required for maximum throughput performance. The adapter is backward compatible with USB 2.0 (limited to 480 Mbps).
  • WiFi 6E Router: To utilize the 6 GHz band and achieve maximum speeds, a WiFi 6E compatible router or access point is required.
  • Antenna Upgrade: RP-SMA female connectors allow for optional higher-gain antenna upgrades to extend range and improve signal quality.
  • Driver Installation: Always download the latest drivers from the official ALFA Network website for optimal performance and compatibility.
