USB NFC Reader III - 13.56MHz

Advanced NFC Forum-certified reader with SAM slot security, supporting all three NFC modes for comprehensive contactless applications. Features firmware upgradeability and enhanced security protocols including FIDO authentication.

NFC Forum Certified SAM Security Firmware Upgradeable FIDO Support Up to 424 kbps

Three Modes of NFC

Complete NFC functionality in one device

Reader/Writer Mode

Read and write data to NFC tags and contactless cards with high-speed access up to 424 kbps for efficient data management and transactions.

Card Emulation Mode

Emulate contactless smart cards enabling the device to act as a virtual card for secure payment and access control applications.

Peer-to-Peer Mode

Enable direct communication between two NFC devices for file exchange, data sharing, and interactive applications with seamless connectivity.

Advanced Features

Enterprise-grade capabilities for professional applications

SAM Security Slot

ISO 7816-compliant SAM slot for key diversification and mutual authentication, providing enhanced security in contactless transactions.

FIDO Authentication

Support for FIDO protocols enabling secure, passwordless login and strong 2FA authentication compatible with PocketKey and YubiKey devices.

Firmware Upgradeable

Post-deployment USB firmware upgrade capability without additional hardware modification, ensuring long-term device adaptability.

Universal Compatibility

CCID and PC/SC compliant plug-and-play USB device ensuring seamless interoperability across platforms and applications.

Programmable Indicators

User-controllable bi-color LED and buzzer for customizable visual and audio feedback during operations and transactions.

Multi-Protocol Support

Comprehensive support for ISO 14443 Type A & B, MIFARE, MIFARE Plus, MIFARE DESFire, FeliCa, and all NFC tag types.
