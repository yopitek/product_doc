Important Information

  • USB 3.0 Recommended: A USB 3.0 port is recommended for optimal performance. The adapter is backward compatible with USB 2.0.
  • WiFi 6 Router: To achieve maximum WiFi 6 speeds, a WiFi 6 (802.11ax) compatible router or access point is required.
  • Antenna Upgrade: RP-SMA connectors allow for optional higher-gain antenna upgrades to extend wireless range.
  • Driver Installation: Always download the latest drivers from the official ALFA Network website for optimal performance and compatibility.
